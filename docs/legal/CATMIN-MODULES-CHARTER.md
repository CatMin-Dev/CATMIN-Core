# Charte Modules Tiers CATMIN

## Règles minimales
- manifest valide
- compatibilité versions explicitée
- dépendances explicites
- pas de contournement sécurité core
- pas d'usurpation de marque CATMIN

## Distribution
- modules tiers possibles
- modules payants possibles si non présentés comme "CATMIN Core"
- obligation de transparence sur l'auteur du module

## Sécurité
- ne pas désactiver protections CSRF/session/headers
- ne pas écrire hors périmètre runtime autorisé
