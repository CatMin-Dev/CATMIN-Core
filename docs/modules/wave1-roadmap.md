# CATMIN Modules Wave 1 Roadmap (055)

## Wave 1 modules (order)
1. Module CMS (pages/contenu)
2. Module SEO/metadata
3. Module media/assets

## Module baseline structure
- `manifest.json`
- `migrations/`
- `controllers/`
- `views/`
- `permissions/`
- `settings/`
- `i18n/`

## Integration targets
- Module manager integration
- Topbar bridge if needed
- Notifications bridge if useful

## Test matrix per module
- Install
- Activate
- Deactivate
- CRUD
- Permissions
- Settings

## Release policy
- Build ZIP test package
- Validate manually end-to-end
- Publish to `CATMIN-Modules` only after explicit human approval

## Versioning
- `MAJOR.MINOR.PATCH-STAGE.NUMBER`
