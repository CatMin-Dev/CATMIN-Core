# Installation CATMIN

## Prerequis
- PHP 8.3+
- extensions requises validees par precheck
- acces ecriture runtime (`storage`, `cache`, `logs`, `sessions`, `tmp`)

## Procedure
1. Ouvrir `/install`.
2. Passer les etapes precheck/legal/template/database/identity/superadmin/security/system.
3. Executer l'installation.
4. Verifier le rapport final.
5. Acceder a `/admin/login`.
