# V2 256 - Email Templates (Hardcore)

## Mission Output
Implement dynamic and configurable template system for production messaging.

## Requirements
- Template CRUD in admin.
- Dynamic variable rendering with validation.
- Preview mode with sample payload.
- Version-safe updates and rollback option.

## Variable Model
- Whitelisted placeholders only.
- Context-specific variable catalogs.
- Safe rendering when variable missing.

## Admin UX
- Live preview (desktop/mobile-like width).
- Subject and body validation.
- Test-send from editor.

## Result
Emails become configurable by operations without breaking delivery consistency.
