# V2 246 - Invoice System (Hardcore)

## Mission Output
Deliver complete invoice system connected to orders.

## Features
- Generate invoice from order.
- HTML and PDF output.
- Invoice lifecycle states (draft/issued/void).
- Immutable issued document policy.

## Integration
- Link invoice records to orders and customers.
- Trigger invoice generation on configured order states.

## Controls
- Unique sequential invoice numbering.
- Tax/total consistency checks.
- Audit logs on generation/void actions.

## Tests
- Generation correctness test.
- Numbering uniqueness test.
- PDF rendering sanity test.

## Result
Invoice system is robust and connected to shop workflow.
