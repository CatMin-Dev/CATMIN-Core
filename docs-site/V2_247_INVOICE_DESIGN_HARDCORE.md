# V2 247 - Invoice Design (Hardcore)

## Mission Output
Produce professional invoice template and visual standards.

## Design Goals
- Clear legal and financial readability.
- CATMIN branding consistency.
- Strong print/PDF legibility.

## Template Structure
- Header (issuer/client/invoice metadata).
- Item table with totals and taxes.
- Payment/notes/footer block.

## Technical Constraints
- Single source template for HTML and PDF.
- Stable typography and spacing across renderers.
- Locale-ready date/number formatting.

## Validation
- Visual review on multi-item and long-description invoices.
- A4 print fidelity check.

## Result
Invoice output looks professional and remains reliable across channels.
