# Mailer Module Base

Le module Mailer V1 fournit une base simple et exploitable:

- configuration d'envoi (driver/from/reply-to/activation)
- templates d'e-mails reutilisables
- historique minimal des envois

## Limites V1

- pas de plateforme newsletter complete
- pas de pipeline avance

## Evolution

La V2 visera l'integration d'une queue pour les envois asynchrones et resilients.
