# V2 267 - Help Center By Module

## Objective
Evolve embedded docs into contextual module-based admin help.

## Scope
- Link docs to module identities.
- Build help navigation per module.
- Surface useful context help in dashboard pages.

## Module Mapping
- One doc index per module.
- Optional quick links from module admin pages.
- Fallback to global docs when module docs missing.

## Maintainability
- Keep simple mapping conventions.
- Avoid hardcoding large content in templates.

## Result
Admin users access relevant help where they work, reducing support friction.
