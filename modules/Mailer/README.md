# Mailer Module

Module Mailer CATMIN: base V1 pour l'envoi systeme et la preparation du mailing futur.

## Portee V1

- configuration simple d'envoi (driver, from, reply-to, activation)
- gestion de templates (code, sujet, contenu html/texte)
- historique minimal des envois (table de traces)

## Hors scope V1

- campagne newsletter complete
- orchestration avancee
- segmentation avancee

## Evolution prevue

- integration d'une queue pour fiabiliser et deferer les envois
- branchement sur des evenements systeme
- enrichissement du suivi des statuts
