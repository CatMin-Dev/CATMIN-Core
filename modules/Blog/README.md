# Blog Module

Module Blog CATMIN: articles editoriaux long format.

## Distinction Blog vs News

- Blog: publication plus narrative, argumentee et detaillee.
- News: information courte, factuelle, rythme rapide.

## Base V1

- titre, slug, extrait, contenu
- statut, date de publication
- liens optionnels vers Media (`media_asset_id`) et SEO (`seo_meta_id`)
- structure extensible: `taxonomy_snapshot` reserve l'evolution vers categories/tags
- interface admin CRUD simple
