# Blocks Module

Base V1 pour des blocs/widgets reutilisables.

## Fonctionnalites

- CRUD admin simple des blocs
- placeholder frontend: `{{ block:slug }}`
- injection dans le contenu des pages frontend

## Limites V1

- pas de versionning de blocs
- pas de rendu conditionnel avance
- pas d'editeur riche integre
