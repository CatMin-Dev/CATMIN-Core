# CAT-MEDIA-LINK

Bridge média transverse CATMIN.

## Fonctions
- média principal (featured)
- galerie liée
- social image
- index des usages média
- presets de génération image (cover/contain/fit)
- variants auto à l'upload (si activé)
- éditeur de crop manuel (canvas) avec override ou nouvelle variante
- fallback variant -> original configurable

## Routes
- `/modules/media-link`
- `/modules/media-link/upload`
- `/modules/media-link/add-url`
- `/modules/media-link/sync`
- `/modules/media-link/presets/save`
- `/modules/media-link/presets/delete`
- `/modules/media-link/settings/save`
- `/modules/media-link/variants/regenerate`
- `/modules/media-link/variants/manual-crop`
- `/modules/media-link/variants/delete`
