# News Module

Module News CATMIN: actualites courtes et factuelles.

## Difference fonctionnelle avec Blog

- News: informations rapides, ponctuelles, format concis.
- Blog: contenu editorial plus long, narratif et approfondi.

## Base V1

- titre, slug, resume, contenu
- statut, date de publication
- liens optionnels vers Media (`media_asset_id`) et SEO (`seo_meta_id`)
- interface admin CRUD simple (pattern CATMIN)
